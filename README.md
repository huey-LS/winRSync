# rsync
sublime text 3 plugin for windows

## version
beta

## Dependencies
 - cwRsync
 - git

## Usage
 - 安装 `cwRsync`
 - 把 `cwRsync` 路径添加到环境变量中
 - 安装 `git`
 - 创建 `ssh-keygen`
 - 环境变量中添加 HOME 值为 %HOMEDRIVE%%HOMEPATH%
 - 配置你的 winRSync 设置
 - 开始使用


## Options
- `winrsync.hosts`
- `winrsync.local_path`
- `winrsync.excludes`
- `winrsync.remote_is_master`
- `winrsync.use_ssh`
- `winrsync.check_remote_git`
- `winrsync.delete_slave`
